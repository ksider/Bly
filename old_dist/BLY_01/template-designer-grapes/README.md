# GrapesJS Badge Builder

Визуальный конструктор бейджей на базе [GrapesJS](https://grapesjs.com/).

## Запуск

```bash
npm install    # установить зависимости (первый запуск)
npm run dev    # открыть http://localhost:5173
```

- Доступны все стандартные блоки GrapesJS (через `grapesjs-blocks-basic`), плюс наши «Поле данных» и «Фигура» в категории *Badge*.
- Холст зафиксирован в размере 90×55 мм, поэтому любые блоки автоматически подстраиваются под формат бейджа.
- Двойной клик по блоку — редактирование текста, стили изменяются через правые панели GrapesJS.
- Для динамических полей используйте блок «Поле данных» и задайте атрибут `data-field` (он превратится в `{{field}}`).
- Кнопка «Экспортировать» выдаёт HTML, CSS и manifest JSON, совместимые с основным приложением Bly.

## Структура

- `src/main.js` — инициализация GrapesJS, кастомные блоки, экспорт.
- `src/style.css` — обвязка интерфейса вокруг редактора.
- `index.html` — точка входа Vite.

При сборке (`npm run build`) готовый статический бандл лежит в `dist/`.
